## A Lab Automation System to Auto-measure the Electro-optical Properties of Liquid Crystal Cell

### Technical solutions

Used Py-visa to control the lab equipment and automated the measurement process.

Used PyQt6 to design the user interface.

Used python and Raspberry Pi Pico to control the motor. The motor can rotate the Liquid Crystal Cell.

### Known issues
Only compatible with AFG-1022 and Keysight InfiniiVision 2000 X-Series Oscilloscope.

Work will with: Python 3.9.13, PyQt 6.4.0.

Still need to test other version.
